# 🐾 Pet Manager Pro

Un sistema interactivo en consola para la gestión de inventarios de mascotas, diseñado con un enfoque en la **experiencia de usuario (UX)** y la **integración de datos robusta**.

## 🚀 Características
- **Validación Total:** Manejo de errores para evitar cierres inesperados (ValueError, IndexError).
- **Interfaz Intuitiva:** Menús dinámicos para gestión de gatos y perros.
- **Herramientas de Búsqueda:** Localización de registros por índice o por nombre.
- **Formateo Estético:** Funciones de visualización "Pretty" para una salida limpia en consola.

## 🛠️ Tecnologías Usadas
- **Python 3.x**: Lógica principal y manejo de estructuras de datos.
- **Módulo Random**: Para la generación de interacciones dinámicas.
